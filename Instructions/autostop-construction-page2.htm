Wiring up the coin sensors

Find the picture below that matches your coin mech. Make sure the the wiring connecter on the end looks the same. Wire colors may be different, but there must be the same number of wires. Click on the picture if you would like a larger view.

A word of caution. Use the diagrams below as a 'best guess'. You should always check the points with a multimeter. We've found 1 machine so far that use 24v for the sensors on the coin mech. If you happen to wire 24v to the modchip, you *will* ruin the chip, and quite possibly your main pachislo cpu board. When using the diagrams below, 'wire 1' is always on the top / left side of the connector as it plugs into the game board socket.

Aruze / Eleco / Macy (9 wires)

Wire pin 2 on the coin mech (white) to pin 3 on the modchip.

Wire pin 3 on the coin mech (green) to pin 2 on the modchip.

Wire pin 2 (yellow) on the game start lever connector to pin 4 on the modchip.

Sammy / Rodeo (6 wires + 2 for electromagnet)

Wire pin 2 on the coin mech (yellow) to pin 2 on the modchip.

Wire pin 5 on the coin mech (white) to pin 3 on the modchip.

 Pin 4 wiring is not available on this model yet. (You can try the brown wire coming off the game start lever. Works on some models, but not others.)

Yamasa (5 wires + 2 for electromagnet)

Wire pin 2 (red) on the coin mech to pin 2 on the modchip.

Wire pin 1 (red) on the coin mech to pin 3 on the modchip.

Wire pin 3 (orange) from the game start lever connector to pin 4 on the modchip.

 

EMA (4 wires + 2 for electromagnet)

Wire pin 3 (yellow) on the coin mech to pin 2 on the modchip.

Wire pin 4 (green) on the coin mech to pin 3 on the modchip.

Wire pin 1 (red) from the game start lever connector to pin 4 on the modchip.

IGT (4 wires + 2 for electromagnet) - version 1+ only

Wire pin 3 (orange) to pin 2 on the modchip.

Wire pin 2 (red) to pin 3 on the modchip.

Wire pin 1 (brown) from the game start lever connector to pin 4 on the modchip.

Thanks to Dritz for the coin mech pictures and wiring info.

Olympia (4 wires + 2 for electromagnet)

Wire pin 2 (brown) to pin 2 on the modchip.

Wire pin 3 (yellow) to pin 3 on the modchip.

Wire pin 4 (black) to pin 8 on the modchip. This creates a common ground.

Wire pin 5 (yellow) from the game start lever connector to pin 4 on the modchip.

Thanks to Dritz for the coin mech pictures and wiring info.

Notes:

We're tested this chip in many types of machines: Aruze, Eleco, Macy, Takasago, Yamasa, Rodeo, Sammy, Heiwa, Daito. The credit add function wasn't able to be performed on a Daito machine (Yeti), because the machines used 24v on the coin sensors. The credit add function wasn't able to work on some Takasago machines (Rocky, Osanpo Tengoku), because the machines use 3 senors in the coin mech.
